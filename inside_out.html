INSIDE OUT
Movie Script Text Analytics

This project involves analyzing dialogue text. Dialogue text is pervasive in our daily lives, showing up in forums, chat, or phone.
Text data can be very rich, but the unstructured nature can be challenging. In this project we apply various text analytics
techniques to gain insight on the movie script of Disney's Inside Out. Can we detect character's emtions through text? This page
presents highlights of our findings. For the comprefensive report and code, please visit the github page. All rights reserved.


Team: Steven Lin, Xiang Li
